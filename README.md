## Task Title
Operating System Security Fundamentals (Linux & Windows)

## Objective
The objective of this task is to understand operating system level security concepts
and apply basic OS hardening techniques to reduce the attack surface of a system.

## Tools Used
- Ubuntu Linux (VM) / Windows Defender
- Alternatives: Kali Linux, Fedora
- VirtualBox (for Linux VM)

## Concepts Covered
- Operating System Security
- OS Hardening
- User Accounts and Access Control
- Administrator vs Standard User
- Linux File Permissions (Read, Write, Execute)
- Least Privilege Principle
- Firewall Configuration (UFW / Windows Firewall)
- Running Processes and Services
- Disabling Unnecessary Services
- OS Security Checklist

## Key Learnings
- Importance of securing the operating system
- How user permissions protect system resources
- Why least privilege principle is critical
- How firewalls help in network security
- How disabling unused services reduces attack surface

## Deliverables
- OS security checklist document (PDF)
- Explanation of OS security fundamentals

## Outcome
This task helped in building a strong understanding of OS-level security,
system hardening practices, and how to protect Linux and Windows systems
from common security threats.
